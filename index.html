Изменения в HTML

1. Тег <html>

Было

<html lang='en'>

Стало

<html lang="ru">

Что поменялось и зачем

Язык страницы меняется с en на ru, потому что весь сайт у тебя на русском.
Это важно для:

  • корректной работы браузера;
  • лучшего чтения страницы поисковыми системами;
  • правильной автопроверки орфографии и доступности.

Стало

    <meta name="viewport" content="width=device-width, initial-scale=1.0"/>

    <script src="https://cdnjs.cloudflare.com/ajax/libs/animejs/3.2.1/anime.min.js"></script>

Что именно добавилось

2.1. Подключение шрифта Montserrat

<link href="https://fonts.googleapis.com/css2?family=Montserrat&display=swap" rel="stylesheet">

Это нужно, чтобы сайт выглядел так же, как во второй версии. Без этой строки шрифт может не подгрузиться.

2.2. Подключение anime.js

<script src="https://cdnjs.cloudflare.com/ajax/libs/animejs/3.2.1/anime.min.js"></script>

Это библиотека для анимаций. Во второй версии сайта она нужна для:

  • плавной смены темы;
  • анимации звёзд;
  • анимации корзины;
  • анимации всплывающих окон.

2.3. Подключение внешнего CSS

<link rel="stylesheet" href="styles.css">

Во второй версии стили вынесены в отдельный файл.
Это значит, что сам HTML становится чище, а оформление живёт отдельно.

Что осталось тем же

  • title не меняется;
  • Font Awesome остаётся;
  • мета-теги остаются, просто чуть аккуратнее записаны.

  • ставит общий шрифт;
  • задаёт цвет текста и фон;
  • делает смену темы плавной.

a{color:inherit}

Ссылки наследуют цвет текста и не выбиваются из общего дизайна.

      x:Math.random()*w,

      ctx.globalAlpha=s.a;

      ctx.fillStyle='#fff';
      ctx.arc(s.x,s.y,s.r,0,Math.PI*2);
      ctx.fill();
    }
  }

  function animate(){
    if (window.anime) {
      for (const s of stars) anime({

        a:[{value:0.15,duration:s.t},{value:1,duration:s.t}],
        easing:'easeInOutSine',
